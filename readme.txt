== Description ==

Infusionsoft Analytics for WordPress is a plugin to inject the Infusionsoft Web Tracking script into every page on your website, without having to edit your theme.

When installed, a new option will be created in the Settings Menu called Infusionsoft Analytics.  In the setup, you will put in your application name and API key.

The plugin will then connect to your Infusionsoft application, get the appropriate JavaScript and then display it on every page of your site when the visitor isn't logged in.


== Installation ==

Here's how to install the plugin and get it working.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Type in your Application Name and API Key into the Settings Page
